Trabajo con Tablas en HTML

Las 3 etiquetas principales para crear una tabla son:

  • table -> Crea la estructura princpial de la tabla
  • tr -> Crea las filas de la tabla
  • td -> Crea cada una de las celdas con datos de la tabla

Otras etiquetas necesarias para trabajar con tablas:

  • th => Crea encabezados de tabla
  • caption => Permite añadir un título de tabla
  • thead => Define las filas que forman el encabezado de la tabla
  • tbody => Define el conjunto de datos de la tabla (sin encabezado ni pie)
  • tfoot => Define la zona de pie de datos de la tabla

Uso de etiquetas de formato de columnas en tablas

En este caso vamos a usar las etiquetas:

  • colgroup => Selecciona el grupo de columnas de la tabla sobre las que aplicar formato
  • col => Define la columna o conjunto de columnas para recibir un formato específico
